# Stadium Safety Inspection Tool

## Overview
A Java-based application designed to track and manage safety inspections in stadiums. Users can log hazards, mark them as resolved, and review inspection records for compliance tracking.

## Features
- **Add Inspections**: Log new safety inspections with descriptions.
- **Resolve Inspections**: Mark inspections as resolved using unique IDs.
- **Display Inspections**: View all inspections with status (resolved/unresolved).
- **Exception Handling**: Prevents invalid inputs and ensures stability.

## Technologies Used
- **Java**  Core development language
- **OOP, Exception Handling, CRUD Operations**
- **List-based Data Storage**

## How to Run
1. Navigate to the project folder:  
   ```sh
   cd stadium-safety-tool
   ```
2. Compile and run:  
   ```sh
   javac *.java && java StadiumSafetyApplication
   ```
